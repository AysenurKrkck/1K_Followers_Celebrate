# 1K_Followers_Celebrate
Bu kod, 1K başarısını kutlamak için bir OLED ekranda animasyonlu kalp ve eşzamanlı yanıp sönen LED efektlerini yönetir. / This code manages an animated heart and synchronized LED effects on an OLED display to celebrate reaching 1K followers.

# 1K Celebration: OLED & LED Animation Project 🚀

[TR] Bu proje, Arduino ve SSD1306 OLED ekran kullanarak 1K takipçi başarısını kutlamak için tasarlanmış bir animasyon sistemidir. Sistem, görsel bir "1K!" metni, boyut değiştiren bir kalp animasyonu ve senkronize LED efektlerini içerir.

[EN] This project is an animation system designed to celebrate the 1K follower milestone using an Arduino and an SSD1306 OLED display. The system includes a visual "1K!" text, a pulsating heart animation, and synchronized LED effects.

## 🛠 Kullanılan Teknolojiler / Technologies Used
- **Mikrodenetleyici / Microcontroller:** Arduino
- **Ekran / Display:** SSD1306 128x64 OLED (I2C)
- **LED:** 4x LED
- **Kütüphaneler / Libraries:** - `U8g2` (OLED ekran kontrolü / OLED display control)
  - `Wire` (I2C haberleşmesi / I2C communication)

## ⚡ Özellikler / Features
- **Kalp Animasyonu / Heart Animation:** Ekranda ritmik olarak büyüyüp küçülen kalp simgesi. / Rhythmic pulsating heart icon on the screen.
- **LED Efektleri / LED Effects:** 200ms aralıklarla sırayla yanan LED dizisi. / Sequence of LEDs blinking at 200ms intervals.
- **Takipçi Kutlaması / Follower Celebration:** 1K başarısını vurgulayan büyük yazı tipi. / Large font emphasizing the 1K milestone.

## 📋 Kurulum / Setup
1. **Arduino IDE**'yi açın. / Open the Arduino IDE.
2. **Kütüphane Yöneticisi**'nden `U8g2` kütüphanesini kurun. / Install the `U8g2` library from the Library Manager.
3. Devre bağlantılarını yapın (SDA/SCL pinlerini OLED için I2C pinlerine bağlayın). / Make the circuit connections (connect SDA/SCL pins to I2C pins for the OLED).
4. Kodu Arduino kartınıza yükleyin. / Upload the code to your Arduino board.
